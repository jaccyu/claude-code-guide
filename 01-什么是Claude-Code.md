# 01 什么是 Claude Code
## 一、基础定义
Claude Code 是 Anthropic 官方推出的 AI 编程助手，深度嵌入 VSCode 编辑器，
主打「一键写代码、改代码、排错、做项目、写文档」。

## 二、官方权威地址
- 官方首页：https://code.claude.ai
- 完整官方文档：https://docs.anthropic.com/claude-code

## 三、核心能力
1. 快速生成 HTML / CSS / JS 网页代码
2. 优化样式、修复bug、精简代码
3. 搭建 GitHub 项目、写README、整理教程
4. 零基础友好，适合运营、文职、新手入门

## 四、适合人群
- 新媒体运营（做作品集、活动页、小工具）
- 零基础想入门编程的新手
- 需要快速产出项目、文档、演示页的职场人
