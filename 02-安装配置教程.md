# 02 Claude Code 完整安装配置教程
## 一、必备软件下载
1. VSCode 代码编辑器（免费）
官方下载地址：https://code.visualstudio.com/

## 二、Claude Code 插件安装
1. 打开 VSCode → 左侧「扩展」
2. 搜索：Claude Code
3. 作者：Anthropic 官方
4. 点击安装

官方插件直达链接：
https://marketplace.visualstudio.com/items?itemName=anthropic.claude-code

## 三、登录激活
1. 安装完成后，左侧出现 Claude 图标
2. 点击登录，绑定 Anthropic 账号
3. 免费版即可满足日常学习、做网页、写代码

## 四、基础使用前提
- 网络正常
- 登录官方账号
- 新建文件夹/文件即可开始AI对话生成代码
